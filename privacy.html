Privacy Policy

This Privacy Policy describes how this application handles user data.

Data Collection

This application does not collect, store, or share personal user data on its own servers.

Use of TikTok API

This application uses TikTok official APIs to upload and manage video content on behalf of the user.

Access to TikTok accounts is granted only after user authorization via OAuth.

Data Usage

The application only uses the minimum required data to perform video uploading and related actions.

No data is sold, shared, or used for advertising purposes.

Third Parties

This application does not share user data with any third parties.

Security

All authentication is handled through TikTok's official authorization system.

Contact

If you have any questions, please contact the developer.
